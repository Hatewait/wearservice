Детали

Веар Сервис предлагает широкий ассортимент износостойких деталей, которые производятся из лучших мировых марок износостойких сталей. В нашем каталоге вы найдете самые типичные применения износостойких деталей для различных отраслей промышленности

Бетоносмесители

Износостойкая броня, заслонки, рычаги, скребки и лопасти

Дорожно-строительная

Детали для ковшей, гидромолотов, отвалов, рыхлителей

Вторичная переработка

Ножи для измельчения, футеровка, детали многочелюстных захватов

Дробильно-сортировочная

Детали для разного типа дробилок, грохота, питателей, мельниц

Прочие изделия

Стрелковые объекты, футеровка, детали многочелюстных захватов
